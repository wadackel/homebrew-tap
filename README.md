# Homebrew Tap for wadackel's Tools

This is a [Homebrew](https://brew.sh/) tap for wadackel's command-line tools.

## Installation

First, tap this repository:

```bash
brew tap wadackel/tap
```

Then install any of the available tools:

```bash
brew install ofsht
```

## Available Tools

### ofsht

Command-line tool for managing Git worktrees with automation features.

- **Repository**: [wadackel/ofsht](https://github.com/wadackel/ofsht)

## Updating

To update all installed tools to the latest versions:

```bash
brew update
brew upgrade
```

## Uninstalling

To uninstall a tool:

```bash
brew uninstall ofsht
```

To remove this tap:

```bash
brew untap wadackel/tap
```
